# AI配音Web版

本项目为AI配音桌面版，基于PyQt开发，实现一键配音、生鲜转换等功能。

## 功能简介

- AI对字幕进行角色标注：准确率90%以上
- 一键生成配音视频，支持克隆声音和预设声音
- 支持多种模型:elevenlabs、cosyvoice

## 安装与运行

1. 克隆本仓库：
    ```bash
    git clone https://github.com/yourusername/AIDubbing-QT-main.git
    ```
